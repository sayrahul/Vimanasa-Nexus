## Available Templates

### 1. Letterhead Template (`letterhead-template.html`)
A blank letterhead template that can be used for any type of document. Simply add your content in the designated content area.

**Usage:**
- Open the file in a web browser
- Replace the content in the `#document-content` section
- Print or save as PDF

### 2. Offer Letter Template (`offer-letter-template.html`)
Professional offer letter template for new hires.

**Fields to customize:**
- Candidate Name
- Address
- Job Title
- Department
- Date of Joining
- Reporting Manager
- Work Location
- Annual CTC
- Probation Period
- Notice Period
- Benefits
- Authorized Signatory details

### 3. Joining Letter Template (`joining-letter-template.html`)
Confirmation letter for employees joining the organization.

**Fields to customize:**
- Employee Name
- Address
- Employee ID
- Job Title
- Department
- Date of Joining
- Reporting Manager
- Work Location
- Working Hours
- First day instructions
- Authorized Signatory details

### 4. Invoice Template (`invoice-template.html`)
Professional invoice template for billing clients.

**Fields to customize:**
- Invoice Number
- Invoice Date
- Due Date
- Client Name and Address
- Service/Product descriptions
- Quantities and Rates
- GST percentage
- Payment terms

## How to Use

1. **Open in Browser:** Double-click any HTML file to open it in your default web browser
2. **Edit Content:** Use a text editor to modify the placeholder text (marked with `[brackets]`)
3. **Print/Save as PDF:** Use your browser's print function (Ctrl+P / Cmd+P) and select "Save as PDF"
