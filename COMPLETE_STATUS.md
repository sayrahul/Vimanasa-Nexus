## 📞 Support & Maintenance

### If Issues Occur:
1. Check browser console (F12)
2. Check terminal for server errors
3. Verify Google Sheets API credentials
4. Check `.env.local` file
5. Run `npm install` if modules missing
6. Clear `.next` folder and rebuild

### Common Fixes:
- **404 Error**: Server on port 3001, try http://localhost:3001
- **Data Not Saving**: Check Google Sheets permissions
- **Charts Not Showing**: Verify Recharts installed
- **PDF Not Generating**: Check employee salary data
